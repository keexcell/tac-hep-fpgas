The resource usage dropped significantly going between standard and arbitrary variables in how many LUTs were used and how much time was taken. It almost cut the usage in half, so this change is definitely useful and necessary if we can limit how many bits we're allocating for our variables. The difference between the different arbitary data types is less apparent here.
